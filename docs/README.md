# Documentation Index

This directory contains the working documentation for Open RTT.

- [protocol_spec.md](protocol_spec.md): control block format, ring semantics, and compatibility rules
- [FIRMWARE_INTEGRATION.md](FIRMWARE_INTEGRATION.md): how to integrate the target library into firmware

- [USAGE.md](USAGE.md): day-to-day usage with DSS and general troubleshooting
- [CONTRIBUTING.md](CONTRIBUTING.md): how to contribute to the project
